# 🥗 MealPlanner — Планировщик питания

Веб-приложение для планирования питания на неделю с рецептами и калькулятором калорий.

---

## 📋 Требования

- Node.js **v20** (LTS)
- npm 9+

```bash
nvm install 20
nvm use 20

🚀 Быстрый запуск
# 1. Клонирование
git clone <url-репозитория>
cd mealplanner

# 2. Настройка
cp .env.example .env

# 3. Установка
make install

# 4. Запуск
make dev

Откройте http://localhost:5173

🛠️ Команды
make install    # Установить зависимости
make dev        # Запустить бэкенд + фронтенд
make backend    # Только бэкенд (порт 5000)
make frontend   # Только фронтенд (порт 5173)
make db-reset   # Сбросить базу данных
make clean      # Очистить всё
